# 姨萌萌隐私政策
### 专为女性设计的生理周期记录与健康管理应用

🔒 数据本地存储
🚫 不上传服务器
📱 鸿蒙原生应用
版本：2.0 | 生效日期：2025年12月7日

🛡️ 隐私保护承诺

姨萌萌（以下简称"我们"）深知女性健康数据的私密性与重要性。我们郑重承诺：您在姨萌萌中记录的所有个人健康数据（包括经期记录、症状记录、体温体重等）均完全存储于您的设备本地，不会上传至任何互联网服务器。您的隐私数据始终由您完全掌控。

📋 关于本政策

本隐私政策旨在向您说明我们如何处理您的个人信息。我们恪守以下原则：数据最小化、本地优先、透明公开、用户掌控。请在使用姨萌萌前仔细阅读本政策。

### 一、我们的数据处理原则
1.1 本地存储优先
姨萌萌采用本地优先（Local-First）的数据架构设计。您记录的所有健康数据，包括但不限于经期日期、症状、体温、体重、情绪等信息，均使用鸿蒙OS提供的安全存储机制保存在您的设备本地，不会通过网络传输至我们的服务器或任何第三方服务器。

1.2 数据完全可控
您可以随时在应用内查看、修改或删除您的所有数据。卸载应用将彻底清除所有本地数据。如需备份，您可以使用设备自带的备份功能，数据备份同样不会经过我们的服务器。

### 二、我们收集的信息
2.1 您主动提供的健康数据（仅本地存储）
为提供经期管理、健康记录、智能预测等核心功能，您可以选择记录以下信息：

经期数据：经期开始/结束日期、周期长度、经血量等
症状记录：痛经程度、情绪状态、身体不适等
生理指标：基础体温、体重、睡眠时长等（可选）
个人备注：您添加的任何文字备注
重要说明：上述所有数据均存储于您的设备本地，采用鸿蒙OS安全存储机制加密保护，不会上传至互联网。

2.2 应用运行必要信息
为保障应用正常运行和持续改进，我们可能收集以下匿名化技术信息：

设备信息：设备型号、鸿蒙OS版本（用于兼容性适配）
应用信息：应用版本号、崩溃日志（用于问题排查）
这些信息不包含任何可识别您身份的内容，仅用于改善应用质量。

2.3 我们不收集的信息
我们明确承诺不会收集以下信息：

您的真实姓名、手机号码、身份证号等身份信息
您的地理位置信息
您的通讯录、短信、通话记录
您的照片、文件等媒体内容
您在其他应用中的使用数据

### 三、应用权限说明
姨萌萌可能申请以下设备权限，所有权限均需您主动授权：

通知权限：用于发送经期提醒、排卵期提醒等本地通知
网络权限：仅用于检查应用更新（不传输您的个人数据）
您可随时在系统设置中管理这些权限。关闭权限可能影响相关功能，但不会影响核心的经期记录功能。

### 四、数据安全保障
4.1 技术安全措施
本地加密存储：采用鸿蒙OS提供的安全存储API，数据加密保存
应用沙箱隔离：数据存储于应用专属沙箱目录，其他应用无法访问
无网络传输：健康数据不经过网络，从根本上杜绝传输泄露风险
4.2 您的数据权利
查看权：随时在应用内查看您的所有记录
修改权：随时修改任何已记录的数据
删除权：随时删除部分或全部数据
导出权：支持将数据导出为通用格式（如适用）

### 五、第三方服务
姨萌萌目前不集成任何第三方数据分析、广告追踪或社交分享SDK。如未来版本引入第三方服务，我们将更新本政策并明确告知。

### 六、未成年人保护
姨萌萌的服务主要面向成年女性用户。若您是未满18周岁的未成年人，请在监护人指导下使用本应用。我们建议监护人关注未成年人的应用使用情况，引导其正确认识生理健康知识。

### 七、政策更新
我们可能适时修订本政策。重大变更将通过应用内通知告知您。继续使用姨萌萌即表示您同意更新后的政策。
